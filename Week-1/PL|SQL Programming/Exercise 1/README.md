# Exercise 1: Control Structures in PL/SQL

## Objective

To understand and implement PL/SQL control structures such as loops, conditional statements, and cursors in a banking management scenario.

## Scenarios Implemented

### Scenario 1: Interest Rate Discount

A PL/SQL block is used to iterate through customers and apply a 1% discount to loan interest rates for customers above 60 years of age.

### Scenario 2: VIP Customer Promotion

A PL/SQL block is used to identify customers with account balances greater than $10,000 and update their VIP status.

### Scenario 3: Loan Due Reminder

A PL/SQL block is used to retrieve loans due within the next 30 days and generate reminder messages using DBMS_OUTPUT.

## Concepts Used

* PL/SQL Blocks
* FOR Loops
* Conditional Processing
* UPDATE Statements
* Cursor Iteration
* DBMS_OUTPUT.PUT_LINE
* Transaction Control (COMMIT)

## Files

* `Exercise1.sql` : Contains all PL/SQL solutions.
* `README.md` : Documentation for the exercise.

## Conclusion

This exercise demonstrates the use of PL/SQL control structures to automate banking operations such as interest updates, VIP promotions, and loan reminder generation.
