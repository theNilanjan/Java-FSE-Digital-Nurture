# PL/SQL Banking System Exercises

## Overview

This repository contains PL/SQL exercises and supporting documentation developed as part of database programming and banking system automation coursework.

The repository demonstrates the use of PL/SQL concepts such as:

* Control Structures
* Stored Procedures
* Loops and Conditional Statements
* Database Transactions
* Data Validation
* Banking Operations Automation

## Exercises Included

### Exercise 1: Control Structures

Implementation of PL/SQL blocks for:

* Applying loan interest discounts to senior citizens.
* Promoting eligible customers to VIP status.
* Generating reminders for loans due within 30 days.

### Exercise 3: Stored Procedures

Implementation of stored procedures for:

* Processing monthly interest on savings accounts.
* Updating employee bonuses by department.
* Transferring funds between customer accounts.
